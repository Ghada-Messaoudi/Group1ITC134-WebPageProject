Cygwin Installation Steps

Let's assume that your USB flash drive is on E:

  1. In your FileExplorer, create a folder at E:\PortableApps\CygwinPortable\packages
  2. Download the Cygwin installer executable file (*.exe filename). In a web browser, goto https://www.cygwin.com/install.html . Click on setup-x86_64.exe link to download into your Downloads folder.
  3. In your FileExplorer, copy from your Downloads folder the “setup-x86_64.exe” file to your E:\PortableApps\CygwinPortable directory.
  4. Install Cygwin by clicking upon the “setup-x86_64.exe” file.
  5. In new window “User Account Control” click “Yes”
  6. Click “Next>” on first dialog install window.
  7. On next window, select “Install from Internet”. Then click “Next>”.
  8. In next window, browse or type “E:\PortableApps\CygwinPortable” to be root directory.Then select “All Users (RECOMMENDED)” for install. Then click “Next>”.
  9. On next window, browse or type “E:\PortableApps\CygwinPortable\packages” for Local Package Directory. Then click “Next>”.
  10. In next window, select “Direct Connection” for Select Your Internet Connection. Then click “Next”.
  11. The next window asks for Choose A Download Site. In your web browser, goto https://cygwin.com/mirrors.html . From here, determine the best site to download nearest to your location. For the Seattle Central College (SCC) students, choose Washington: http://mirrors.metapeer.com . Then click “Next>”.
  12. Next page will be downloading status page. When done, the window will close and a new window will open called “Select Packages”.
  13. In “Select Packages” window, click “Next>”.
  14. In next window, ensure/select “Select required packages (RECOMMENDED)”. Then click “Next>”.
  15. Next page will be ANOTHER downloading status page. THIS WILL TAKE ABOUT FIVE (5) TO TEN (10)MINUTES OF TIME WAITING. When done, the window will close and a new window will open called “Create Icons” and “Installation Status”. Make sure NO boxes are selected/checked. Then click “Finish”. This means on the bare-bones default Cygwin packages have been installed.
  16. Now that the Cygwin install is done, you need to create the command-line Terminal Window short-cut on USB Stick. So, in your FileExplorer, in folder “E:\PortableApps\CygwinPortable”, right-click and select “New >” then select “Shortcut”. In “Create Shortcut” window, type “E:\PortableApps\CygwinPortable\bin\mintty.exe -i /Cygwin-Terminal.ico -” . Then click on “Next “. In next window, type in “Cygwin_Linux_Terminal”
  17. HURRAH . . . . . . NOW, you can finally double-click on “Cygwin_Linux_Terminal” icon to launch a Cygwin-Linux Command Line Terminal window with “bash” linux-shell.
  18. Now, keep this Terminal-Window open while we setup gVim text-editor and TkDiff code-debugger tool.

gVim Installation Steps

  1. In a web browser, goto http://www.vim.org/download.html . Click on “self-installing executable” link in order to download gVim installing executable file.
  2. In your FileExplorer, copy from your Downloads folder the “gvim[VERSION_NUMBER].exe” file to your E:\PortableApps\CygwinPortable directory. For this cheatsheet, we will call this file “gvim80-586.exe”.
  3. Install gVim by clicking upon the “gvim80-586.exe” file.
  4. In new window “User Account Control” click “Yes”
  5. In next window, click “Yes”.
  6. In new window, click “I Agree” .
  7. In next window, click “Next >”
  8. In new window, browse or type “E:\PortableApps\CygwinPortable\Vim”. Click on “Install” button.
  9. In “Completed” window, click “Close” button.
  10. In next window, click “No” button.
  11. NOW THAT gVim is installed, we need to perform some Linux functions to make gVim usable in Cygwin-Linux-Terminal.
  12. In Terminal-Window, at command prompt, type the following Linux commands:
    13. cd /
    ln -s /Vim/vim80/gvim.exe /bin/.
  13. ALMOST MOST DONE. Next, you need to add the following lines to end of the gVim “ _vimrc ” runtime config file located at "E:\PortableApps\CygwinPortable\Vim\_vimrc" :
    colorscheme blue
    set guifont=Lucida_Console:h18
    set number
     This will allow gVim to open up with dark-blue background, fixed-width font of size 18, and line numbers shown on the left-side of the gVim text editor window.
  14. Now you can launch gVim text editor. You should Google search “gVim basics” or “gVim tutorial” to run thru some basics. It will seem a little difficult at first toggling between “[ESC]-KEY Command Mode” and “[I]-Key Insert/Edit Mode”. However, once you have spent a day messing around with gVim, you will be able to code with VERY LIMITED MOUSE involvement. Plus, with the added power of regular expression search/replace, you will have major editing upgrade vs. the simple “CTRL-H” Search/Replace in other Windows editors like Notepad, Wordpad, IDLE_for_Python, etc.
  15. To launch gVim, simply goto into your open Cygwin_Terminal command line window and type at command prompt “ gvim FILE1 & “ .

TkDiff Installation Steps

  1. In a web browser, goto https://sourceforge.net/projects/tkdiff . Click on green and white “Download” button link in order to download TkDiff installing executable file.
  2. In your FileExplorer, copy from your Downloads folder the “TkDiff[VERSION_NUMBER]-Setup.exe” file to your E:\PortableApps\CygwinPortable directory. For this cheatsheet, we will call this file “TkDiff-4.2.exe”.
  3. Install gVim by clicking upon the “TkDiff-4.2.exe” file.
  4. In new window “User Account Control” click “Yes”
  5. In next window, click “Yes”.
  6. In new window, click “Next >” .
  7. In next window, click “Browse... ” and select/type in “E:\PortableApps\CygwinPortable\TkDiff”. Then, click “Ok”. Then, click “Next >”.
  8. In new window, click “Next >”.
  9. In next window, de-select “View Readme”, “Create Desktop Shortcut”, and “Create Quick Launch Shortcut”. Then, click “Finish”.
  10. NOW THAT TkDiff is installed, we need to perform some Linux functions to make TkDiff usable in Cygwin-Linux-Terminal.
  11. In Terminal-Window, at command prompt, type the following Linux commands:
    cd /
    ln -s /TkDiff/tkdiff.exe /bin/.
  12. Create two text files called FILE1 and FILE2 in “”. The contents to be added for each file is list below with explicit line numbers. You must create these files exactly in order to see quickly the TkDiff effect when diff-ing two text/code files.
    FILE1:
    1 left only, nothing on right side
    2
    3 SAME
    4
    5 left conflict.
    6
    7

    FILE2:
    1
    2 SAME
    3
    4 right conflict.
    5
    6 right only, nothing on left side
  13. Now you can launch TkDiff text/code difference debug tool. You should Google search “TkDiff basics” or “TkDiff tutorial” to run thru some basics. Once you mastered some of the basics of TkDiff, you will be able to even merge two files with conflicts similar to how Git-CMD-SHELL and GitHubDesktop implement conflict resolution.
  14. To launch TkDiff, simply goto into your open Cygwin_Terminal command line window and type at command prompt “ tkdiff FILE1 FILE2 & “ . See that RED means text in FILE1 but not in FILE2. GREEN means text in FILE2 but not in FILE1. BLUE means text in CONFLICT between FILE1 and FILE2. “[N]-KEY” moves to next diff. “[P]-KEY” moves to previous diff. “[F]-KEY” will move to first diff; “[L]-KEY” will move to last diff. Press “[R]-KEY” to refresh TkDiff window. To quit, simple press “[Q]-KEY”.

Now you rock !
